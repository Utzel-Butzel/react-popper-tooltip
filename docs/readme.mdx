## Controlled

_`tooltipShown` prop_

<Playground>
  <StateContainer>
    {({ on, set, toggle }) => (
      <div>
        <button onClick={toggle}>Toggle tooltip</button>
        <br />
        <br />
        <BasicTooltipTrigger
          trigger="none"
          tooltipShown={on}
          onVisibilityChange={set}
          tooltip="Hello, Controlled!"
        >
          I am state controlled!
        </BasicTooltipTrigger>
      </div>
    )}
  </StateContainer>
</Playground>

## Escape to close

<Playground>
  <StateContainer>
    {({ on, set }) => (
      <BasicTooltipTrigger
        trigger="click"
        tooltipShown={on}
        onVisibilityChange={set}
        tooltip={
          <EscapeToClose closeTooltip={() => set(false)}>
            Press Esc to Close
          </EscapeToClose>
        }
      >
        Click Here
      </BasicTooltipTrigger>
    )}
  </StateContainer>
</Playground>

_Note: `StateContainer` used in the snippet above is for demo purpose only and used for state management._

## Continuation
